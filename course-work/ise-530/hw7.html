11.13¶

A function \(f: \mathbb{R}^n \rightarrow \mathbb{R}\) is said to be separable if it can be written as the sum of functions of a single real variable, that is

\begin{aligned} f(x_1, x_2, \cdots , x_n) &= \sum^{n}_{j=1} f_j (x_j). \ \end{aligned}

(a) Let \(f\) be a differentiable separable function and consider the nonlinear program

\begin{aligned} \text{minimize } & f(x_1, x_2, \cdots , x_n) \ \text{subject to } & \sum^{n}_{j=1} x_j = \sigma \ x_j \geq 0, j = 1, \cdots, n, \end{aligned}

where \(\sigma\) is a positive scalar. Write the first-order conditions for a local minimizer \(\bar{x}\) in this nonlinear program.

Lagrangian:

\begin{aligned} \mathcal{L}(x, \mu) &= \sum^{n}{j=1} f_j (x_j) + \mu\Big(\sum^{n}{j=1} x_j - \sigma\Big) \ \end{aligned}

First Order conditions for a local minimizer \(\bar{x}\) in this nonlinear program:

\begin{aligned} \begin{bmatrix} f^\prime_1 (\bar{x}_1) \ f^\prime_2 (\bar{x}_2) \ \vdots \ f^\prime_n (\bar{x}_n) \ \end{bmatrix} &= 0 \ \end{aligned}

\(\mu\) is free.

(b) Recall that if \(x \in \mathbb{R}^n\), the set \(\{j : x_j \not= 0\}\) is called the support of \(x\), denoted by \(\sup(x)\). Show that every feasible solution in part (a) has a nonempty support.

Since we know that \(\sigma\) is a positive scalar, \(x_j \geq 0\), and a feasible solution to part (a) must satisfy the constraint \(\sum^{n}_{j=1} x_j = \sigma\), one of the \(x_j\)s MUST be positive, hence any feasible solution will have a nonempty support \(\blacksquare\).

(c) Show that there is a real number \(\kappa\) such that if \(x^∗\) is a local minimizer of the nonlinear program in part (a), then for all \(j \in \sup(x^∗)\)

\begin{aligned} \frac{\partial f_j (x^*_j)}{\partial x_j} &= \kappa. \ \end{aligned}

As shown in the first order conditions, the partial derivative (simply the derivative since the function is of just one variable), must be 0. Hence \(\kappa = 0\).

Find all extremal values of \(z\), and the corresponding values of \(x_1\) and \(x_2\), using the method of Lagrange multipliers in the following two instances.

(a)

\begin{aligned} x_1x_2 + 4 &= z \ x^2_1 + x^2_2 &= 8. \ \end{aligned}

Finding infimum of Lagrangian:

\begin{aligned} \nabla_x \mathcal{L}(x_1, x_2, \lambda) &= 0 \ \nabla_x \Big(x_1x_2 + 4 + \lambda(x^2_1 + x^2_2 - 8)\Big) &= 0 \ \begin{bmatrix} x_2 + 2\lambda x_1 \ x_1 + 2\lambda x_2 \ \end{bmatrix} &= \begin{bmatrix} 0 \ 0 \ \end{bmatrix} \ \newline \ \text{Substituting }x_1 = -\frac{x_2}{2\lambda}, & \ -\frac{x_2}{2\lambda} + 2\lambda x_2 &= 0 \ 2\lambda x_2 &= \frac{x_2}{2\lambda} \ 4\lambda^2 x_2 &= x_2 \ \lambda &= \pm\frac{1}{2} \ \end{aligned}

Case I: \(\lambda = \frac{1}{2} \rightarrow x_1 = -x_2\)

\begin{aligned} x^2_1 + x^2_2 &= 8 \ (-x_2)^2 + x^2_2 &= 8 \ 2x^2_2 &= 8 \ x^2_2 &= 4 \ x_2 &= \pm2 \text{ and } x_1 = \mp2, z = 0 \ \end{aligned}

Case II: \(\lambda = -\frac{1}{2} \rightarrow x_1 = x_2\)

\begin{aligned} x^2_1 + x^2_2 &= 8 \ (-x_2)^2 + x^2_2 &= 8 \ 2x^2_2 &= 8 \ x^2_2 &= 4 \ x_2 &= \pm2 \text{ and } x_1 = \pm2, z = 8 \ \end{aligned}

(b)

\begin{aligned} x_1 + 2x_2 &= z \ x^2_1 + x^2_2 &= 13. \ \end{aligned}

Finding infimum of Lagrangian:

\begin{aligned} \nabla_x \mathcal{L}(x_1, x_2, \lambda) &= 0 \ \nabla_x \Big(x_1 + 2x_2 + \lambda(x^2_1 + x^2_2 - 13)\Big) &= 0 \ \begin{bmatrix} 1 + 2\lambda x_1 \ 2 + 2\lambda x_2 \ \end{bmatrix} &= \begin{bmatrix} 0 \ 0 \ \end{bmatrix} \ x^* &= -\frac{1}{\lambda}\begin{bmatrix} \frac{1}{2} \ 1 \ \end{bmatrix} \end{aligned}

Primal Feasibility:

\begin{aligned} x^2_1 + x^2_2 &= 13 \ (-\frac{1}{2\lambda})^2 + (-\frac{1}{\lambda})^2 &= 13 \ \frac{1}{4\lambda^2} + \frac{1}{\lambda^2} &= 13 \ \frac{5}{4\lambda^2} &= 13 \ 5 &= 52\lambda^2 \ \lambda &= \pm\sqrt{\frac{5}{52}} \ \end{aligned}

Case I: \(\lambda = \sqrt{\frac{5}{52}} \rightarrow x^* = -\sqrt{\frac{52}{5}}\begin{bmatrix} \frac{1}{2} \\ 1 \\ \end{bmatrix}\)

\begin{aligned} x_1 &= -\frac{1}{2}\sqrt{\frac{52}{5}} \ x_2 &= -\sqrt{\frac{52}{5}} \ z &= -\frac{5}{2}\sqrt{\frac{52}{5}} \ \end{aligned}

11.32¶

Consider the following univariate optimization problem

\begin{aligned} \text{minimize } & x^2 \ \text{subject to } & x^2 - 5x + 6 \leq 0. \ \end{aligned}

(a) Form the Lagrangian.

Lagrangian function:

\begin{aligned} \mathcal{L}(x, \lambda) &= x^2 + \lambda(x^2 - 5x + 6) \ \end{aligned}

(b) Find the dual of the problem.

Lagrangian Dual Function:

\begin{aligned} g(\lambda) &= \underset{x}{\text{inf}} \mathcal{L}(x, \lambda) \ &= \underset{x}{\text{inf}}\Big(x^2 + \lambda(x^2 - 5x + 6)\Big) \ \end{aligned}

Dual Problem:

\begin{aligned} \underset{\lambda}{\text{maximize }} & g(\lambda) \ \text{subject to } & \lambda \geq 0. \ \end{aligned}

Finding infimum of Lagrangian:

\begin{aligned} \nabla_x \mathcal{L}(x, \lambda) &= 0 \ \nabla_x \Big( x^2 + \lambda(x^2 - 5x + 6) \Big) &= 0 \ 2x + \lambda(2x - 5) &= 0 \ x^* &= \frac{5\lambda}{(2 + 2\lambda)} \ \end{aligned}

Lagrangian Dual Function:

\begin{aligned} g(\lambda) &= (x^)^2 + \lambda((x^)^2 - 5x^* + 6) \ &= \Big(\frac{5\lambda}{(2 + 2\lambda)}\Big)^2 + \lambda\Big(\Big(\frac{5\lambda}{(2 + 2\lambda)}\Big)^2 - 5\frac{5\lambda}{(2 + 2\lambda)} + 6\Big) \ &= \lambda\frac{24 + 23\lambda - \lambda^2}{(2 + 2\lambda)^2} \ \end{aligned}

Substituting into Dual Problem:

\begin{aligned} \underset{\lambda}{\text{maximize }} & \lambda\frac{24 + 23\lambda - \lambda^2}{(2 + 2\lambda)^2} \ \text{subject to } & \lambda \geq 0. \ \end{aligned}

Supremum of Dual function:

\begin{aligned} \nabla_\lambda g(\lambda) &= 0 \ \nabla_\lambda \lambda\frac{24 + 23\lambda - \lambda^2}{(2 + 2\lambda)^2} &= 0 \ \frac{-2(\lambda^3 + 3\lambda^2 - 22\lambda - 24)}{(2+2\lambda)^3} &= 0 \ \frac{-2(0.5\lambda^2 + \lambda - 12)(2+2\lambda)}{(2+2\lambda)^3} &= 0 \ \frac{-\lambda^2 - 2\lambda + 24}{(2+2\lambda)^2} &= 0 \ \frac{-(\lambda + 6)(\lambda - 4)}{(2+2\lambda)^2} &= 0 \ \end{aligned}

Case I: \(\lambda^* = -6\)

Rejected, since \(\lambda \geq 0\) constraint.

Case II: \(\lambda^* = 4\)

Optimal Dual Objective value: \begin{aligned} g(\lambda^) &= \lambda^\frac{24 + 23\lambda^* - (\lambda^)^2}{(2 + 2\lambda^)^2} \ &= 4\frac{24 + 23(4) - 4^2}{(2 + 2(4))^2} \ &= 4\frac{24 + 92 - 16}{(2 + 8)^2} \ &= 4\frac{100}{100} \ &= 4 \ \end{aligned}

(c) Verify the Weak Duality Theorem for this problem.

Primal Objective value:

With \(x^* = \frac{5\lambda}{(2 + 2\lambda)}\) and \(\lambda^* = 4\),

\begin{aligned} f(x^*) &= \Big(\frac{5(4)}{(2 + 2(4))}\Big)^2 \ &= 4 \ \end{aligned}

Since solution to primal (4) is \(\geq\) solution to dual (4), weak duality holds.

(d) Does Strong Duality hold, or is there a duality gap?

Since the duality gap is 0 as optimal objective value for both primal and dual problem is the same, Strong duality holds.

11.36¶

Use the Lagrangian dual process to show that

\begin{aligned} X &= { x \in \mathbb{R}^3: x^2_1 + x^2_2 + x^2_3 \leq 1\text{ and }x_1 + x_2 + x_3 = 2 } \end{aligned}

is an empty set.

Suppose the following maximization problem

\begin{aligned} \underset{x_1,x_2,x_3}{\text{maximize }} & x^2_1 + x^2_2 + x^2_3 \ \text{subject to } & x_1 + x_2 + x_3 = 2. \ \end{aligned}

Lagrangian:

\begin{aligned} \mathcal{L}(x_1,x_2,x_3,\mu) &= x^2_1 + x^2_2 + x^2_3 + \mu(x_1 + x_2 + x_3 - 2) \ \end{aligned}

First order conditions:

\begin{aligned} \nabla_x\mathcal{L}(x_1,x_2,x_3,\mu) &= 0 \ 2\begin{bmatrix} x_1 \ x_2 \ x_3 \ \end{bmatrix} + 2\mu\begin{bmatrix} 1 \ 1 \ 1 \ \end{bmatrix} &= \begin{bmatrix} 0 \ 0 \ 0 \ \end{bmatrix} \ x^* &= -\mu\begin{bmatrix} 1 \ 1 \ 1 \ \end{bmatrix} \end{aligned}

Primal Feasibility:

\begin{aligned} x_1 + x_2 + x_3 &= 2 \ -3\mu &= 2 \ \mu^* &= -\frac{2}{3} \ \end{aligned}

Hence,

\begin{aligned} x^* &= \frac{2}{3}\begin{bmatrix} 1 \ 1 \ 1 \ \end{bmatrix} \ \end{aligned}

This makes the optimal objective value to the maximization problem: \(3\Big(\frac{2}{3}\Big)^2 = \frac{4}{3}\). Hence, since maximium value is not \(> 1\), the intersection of the 2 constraints is an empty set. \(\blacksquare\)

  • Determine the minimum objective value and minimizers of the convex program:

\begin{aligned} \underset{x, y}{\text{minimize }} & e^{-y} \text{ subject to } \sqrt{x^2 + y^2} - x \leq 0, \end{aligned}

Show that the dual function:

\begin{aligned} d(\mu) \overset{\Delta}{=} \underset{x, y}{\text{minimize }} & e^{-y} + \mu \Big[ \sqrt{x^2 + y^2} - x \Big], \mu \geq 0 \end{aligned}

is identically equal to zero but this value is not attained, for any \(\mu \geq 0\). This convex program has a positive gap in this generalized sense. Without worrying about the non-attainment issue of the dual function, what goes wrong with this example with reference to the general duality theory?

Lagrangian:

\begin{aligned} \mathcal{L}(x, y, \lambda) &= e^{-y} + \mu \Big[ \sqrt{x^2 + y^2} - x \Big] \ \end{aligned}

Lagrangian Dual Function:

\begin{aligned} d(\mu) &= \underset{x,y}{\text{inf}} \mathcal{L}(x, y, \mu) \ &= \underset{x,y}{\text{inf}}\Big(e^{-y} + \mu \Big[ \sqrt{x^2 + y^2} - x \Big]\Big) \ \end{aligned}

Finding infimum of Lagrangian:

\begin{aligned} \nabla_{x,y} \mathcal{L}(x,y, \lambda) &= 0 \ \nabla_{x,y} \Big(e^{-y} + \mu \Big[ \sqrt{x^2 + y^2} - x \Big]\Big) &= 0 \ \begin{bmatrix} 0.5\mu(x^2 + y^2)^{-0.5}(2x) -\mu \ -e^{-y} + 0.5\mu(x^2 + y^2)^{-0.5}(2y) \ \end{bmatrix} &= \begin{bmatrix} 0 \ 0 \ \end{bmatrix} \ \begin{bmatrix} \mu x (x^2 + y^2)^{-0.5} -\mu \ -e^{-y} + \mu y (x^2 + y^2)^{-0.5} \ \end{bmatrix} &= \begin{bmatrix} 0 \ 0 \ \end{bmatrix} \ \begin{bmatrix} \mu x \ \mu y \ \end{bmatrix} &= \begin{bmatrix} \mu\sqrt{x^2 + y^2} \ e^{-y}\sqrt{x^2 + y^2} \ \end{bmatrix} \ \newline \ \text{From } \mu x = \mu\sqrt{x^2 + y^2}, & \ x &= \sqrt{x^2 + y^2} \ x^2 &= x^2 + y^2 \ y^2 &= 0 \ y &= 0 \ x &= \text{Any value} \ \end{aligned}

Hence, since \(x\) can take on any value and \(y=0\), the dual function reduces to:

\begin{aligned} d(\mu) &= 1 + (\mu - 1)x \ \end{aligned}

If we choose \(x \rightarrow -\infty\), \(d(\mu) \rightarrow -\infty\) and dual function is unbounded below.

Under Weak Duality Thm, we know that the optimal dual value is a lower bound on the optimal primal value, but since our optimal dual value here is \(-\infty\), duality theory is useless in helping us find the optimal primal value to the problem.
